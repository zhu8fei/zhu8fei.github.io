    Hexo的常用命令

    init a blog

    $ hexo init blog

    本地安装

    $ npm install hexo-cli -g

    init a exist blog

    $ npm install --save

    Create a new post

    $ hexo new "My New Post"

    More info: Writing

    Clean static files

    $ hexo clean

    Generate static files

    $ hexo generate
$ hexo g

    More info: Generating

    Run server

    $ hexo server
$ hexo s

    More info: Server

    Deploy to remote sites

    $ hexo deploy

    More info: Deployment

    uml工具

    mermaid

    cdn : http://www.bootcdn.cn/
    个性修改:

    1. 关闭_config.xml中默认的 highlight.enable=false
    2. 修改after-footer.ejs把jquery拿到head.ejs中. 提前加载防止各种找不到jquery的问题.
    3. 修改head.js
    <% if (config.highlight.enable){ %>
<!-- <link href="//fonts.googleapis.com/css?family=Source+Code+Pro" rel="stylesheet" type="text/css">-->
<!-- WTF GFW 360 mirrors -->
<link href="//fonts.useso.com/css?family=Source+Code+Pro" rel="stylesheet" type="text/css">
<% } %>
<link rel="stylesheet" href="//cdn.bootcss.com/highlight.js/9.8.0/styles/monokai.min.css">
<script src="//cdn.bootcss.com/highlight.js/9.8.0/highlight.min.js"></script>
<script src="//cdn.bootcss.com/highlightjs-line-numbers.js/1.1.0/highlightjs-line-numbers.js"></script>
<link rel="stylesheet" type="text/css" href="//cdn.bootcss.com/mermaid/6.0.0/mermaid.forest.min.css">
<script type="text/javascript" src="//cdn.bootcss.com/mermaid/6.0.0/mermaid.min.js"></script>
    1. footer.ejs添加
      <script type="text/javascript">
$(document).ready(function() {
 // 不要用highlight渲染uml图
 $('pre code').not(".mermaid").each(function(i, block) {
   hljs.highlightBlock(block);
   hljs.initLineNumbersOnLoad();
 });
 // 底色防坑.
 $(".mermaid").parent().css("background","#ffffff");
 // 只渲染mermaid样式的对象.
 mermaid.init({startOnLoad:true},$(".mermaid"));
})
</script>
